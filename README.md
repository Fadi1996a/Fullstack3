# Fullstack1

# to limit docker memory usage
Copy .wslconfig to your C:\Users\[username]

# to run docker
```
docker-compose up
```
if dockerfiles changed run this once
```
docker-compose up --build
```

shut down docker
```
docker-compose down
```
# Run server  and Angular
cd to backend Folder  
```
node server.js
```


cd to Frontend Folder  
```
ng serve
```
